Welcome Candidate

You have been tasked with completing the following user story:

Display a list of customers with togglable details

Description

A user should be able to click on the 'Customer Directory' link where they will be routed to /customer-directory. On the customer directory route the user should see a list of customers returned using a GET request. The details of each customer should be hidden initially. Clicking on a customers card should expand the details for that customer. Clicking again should close the details.

Acceptance Criteria

  • Users can navigate to /customer-directory using the Customer Directory link located on the homepage
  • Users should see a list of customers with their details hidden by default
  • Users should be able to click on a customer to expand or collapse the customer details
  • Follows the provided design as closely as possible considering you can't inspect the design
